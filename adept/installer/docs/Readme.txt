Welcome to the Adept Installer!

Please note that you MUST install Java JDK Version 1.6.0_20 or higher and JavaFX Version 1.3.1 
or higher in order to use this software.

The installer will load Adept & the Image Loader in the Applications/Adept folder. To run the
program, navigate to the Applications/Adept folder and double click on the Adept.command or
 Applications/ImageLoader.command shortcut.

tomlee@ai.sri.com
